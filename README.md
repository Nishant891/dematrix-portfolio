
# Getting Started

## 1. Clone the repository
```bash
git clone https://github.com/Nishant891/dematrix-portfolio/
cd dematrix
````

## 2. Install dependencies

```bash
npm install
```

## 3. Start the development server

```bash
npm run dev
```

---

# Adding Articles

Place your articles as **Markdown (.md)** files inside the `content/` directory.

You can also create subfolders inside `content/` and the system will treat them as categories.
For example:

```
content/
  linux/
    permissions.md
  networking/
    osi-layer.md
  my-article.md
```

Each Markdown file will be automatically detected and rendered.
